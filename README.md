# Kasir

Aplikasi kasir berbasis web yang dibuat sebagai project UKK (pengayaan).

## 🛠️ Teknologi
- HTML
- CSS
- JavaScript
- PHP
- Tailwind CSS

## ✨ Fitur
- Login & autentikasi pengguna
- Transaksi kasir
- Cetak struk / receipt
- Manajemen produk
- Laporan penjualan

## 📁 Cara Menjalankan
1. Clone repo ini
```bash
   git clone https://github.com/esselrn/kasir.git
```
2. Pindahkan folder ke direktori server lokal (contoh: `htdocs` untuk XAMPP atau `www` untuk Laragon)
3. Import database dari folder `/database` (jika tersedia)
4. Jalankan **XAMPP** atau **Laragon**, aktifkan Apache & MySQL
5. Buka browser dan akses `http://localhost/kasir`

## 📝 Catatan
Project ini dibuat untuk keperluan UKK (Uji Kompetensi Keahlian) dan tidak di-deploy secara publik.
